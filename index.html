Decree Extractor

The Decree Extractor is a web application designed to streamline the retrieval and analysis of legislative decrees from El Salvador's official government website. Users can select a specific year to extract decrees, view them in a dynamic table, and analyze issuance dates through interactive graphs. The app also offers options to download the data in Excel and CSV formats for further analysis.

  • R: Primary programming language for data manipulation and application logic.
  • Shiny: Framework for building interactive web applications in R.
  • Rvest: Web scraping tool to extract decree data from the official website.
  • Tidyverse: Collection of R packages for data science, ensuring efficient data manipulation and visualization.
